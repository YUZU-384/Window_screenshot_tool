# Window_screenshot_tool
一个简单的连续截图工具，使用 **易语言** 编写。

---

## 特性
- 指定间隔完成截图任务
- 指定截图的次数（截图数量）
- 支持压缩图片为jpg（质量1-100）
- 支持前台与后台截图（后台部分窗口不适用）

## 安装
确保安装了 **易语言5.x**(4.xx应该也)

## 使用
运行应用："main-Window_screenshot_tool-x.x.x.x.e"

## 作用
你可以使用次工具为yolo获取训练所需的图片
